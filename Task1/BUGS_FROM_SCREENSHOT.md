# BUGS.md

**Дата:** 25 ноября 2025

## 1. Фильтры по направлению не работают (High)

**Описание бага**  
При выборе одного направления показываются вакансии разных направлений.

**Шаги воспроизведения**

1. Перейти на сайт Avito Team
2. Выбрать только одно направление (например, «Data Science»)
3. Дождаться обновления результатов

**Ожидаемое поведение**  
Только вакансии выбранного направления

**Фактическое поведение**  
Появляются вакансии из Продаж, Клиентского сервиса, Финансов, Недвижимости, M&A и др.

**Приоритет** High  
**Обоснование** Полностью ломается основной сценарий поиска вакансий.

## 2. Ложное сообщение «Ничего не нашлось» (High)

**Описание бага**  
Сообщение выводится, хотя ниже по странице есть вакансии.

**Шаги воспроизведения**

1. Применить любой набор фильтров
2. Пролистать вниз

**Ожидаемое поведение**  
Сообщение появляется только при нулевом результате

**Фактическое поведение**  
Сообщение есть, хотя вакансии отображаются

**Приоритет** High  
**Обоснование** Вводит пользователя в заблуждение.

## 3. Фильтр формата работы не применяется (High)

**Описание бага**  
При выборе «Можно удалённо» остаются офисные вакансии.

**Шаги воспроизведения**

1. Выбрать фильтр «Можно удалённо»
2. Пролистать результаты

**Ожидаемое поведение**  
Все вакансии поддерживают выбранный формат

**Фактическое поведение**  
Есть вакансии только с офисом в Москве, без возможности удалёнки

**Приоритет** High  
**Обоснование** Формат работы — один из ключевых фильтров при выборе вакансии.

## 4. При активном фильтре «Команда» выводится сообщение «Вакансий пока нет» (High)

**Описание бага**  
Пользователь включает фильтр «Команда», но вместо списка команд или корректного сообщения появляется текст «Вакансий пока нет».

**Шаги воспроизведения**

1. Выбрать фильтр «Команда»
2. Посмотреть на результаты или панель активных фильтров

**Ожидаемое поведение**

- Либо список доступных команд
- Либо текст «Нет команд для этих вакансий»

**Фактическое поведение**  
Появляется сообщение «Вакансий пока нет»

**Приоритет** High  
**Обоснование** Логическая ошибка: фильтр по командам, а сообщение про вакансии

## 5. Фильтр «Руководящая роль» пропускает обычные позиции (Medium)

**Описание бага**  
При включённой галочке показываются вакансии без признаков руководства.

**Шаги воспроизведения**

1. Поставить галочку «Руководящая роль»
2. Пролистать результаты

**Ожидаемое поведение**  
Только Head/Lead/Руководитель

**Фактическое поведение**  
Обычные «Менеджер», «Аналитик» и т.п.

**Приоритет** Medium  
**Обоснование** Фильтр теряет смысл, но поиск всё ещё возможен вручную.

## 6. Некорректные счётчики вакансий в категориях (Medium)

**Описание бага**  
Цифра в заголовке не совпадает с реальным количеством карточек.

**Шаги воспроизведения**

1. Применить фильтры
2. Посмотреть заголовки («Клиентский сервис 2»)

**Ожидаемое поведение**  
Цифра = количество видимых вакансий

**Фактическое поведение**  
«Клиентский сервис 2» → видна 1 вакансия

**Приоритет** Medium  
**Обоснование** Подрывает доверие пользователя

## 7. Непоследовательное оформление локаций и формата работы (Low)

**Описание бага**  
Разные карточки используют разные формулировки одной информации.

**Шаги воспроизведения**

1. Пролистать любую выдачу

**Ожидаемое поведение**  
Единый стиль во всех карточках

**Фактическое поведение**  
«Москва», «офис и удаленно», «Москва, Санкт-Петербург офис и удаленно», «удалённая работа»

**Приоритет** Low  
**Обоснование** Косметический недочёт, не влияет на функциональность

## Итог

- High: 4 бага — ломают основной сценарий поиска и фильтрации
- Medium: 3 бага — сильно портят UX и доверие
- Low: 1 баг — косметика
