# TESTCASES.md

В этом документе описаны реальные тест-кейсы, выполненные вручную в Postman и автоматизированные далее.  
Все тесты соответствуют фактическому поведению API и опираются только на корректные успешные запросы.

---

## Тест-кейс 1: Успешное создание объявления

**Метод:** POST /api/1/item  
**Тело запроса (пример из реального теста):**

```plaintext
{
  "sellerID": 12345,
  "name": "Burger",
  "price": 300,
  "statistics": {
    "likes": 4,
    "viewCount": 200,
    "contacts": 2
  }
}
```

**Ожидаемый результат:**

- HTTP 200
- В ответе есть строка `"Сохранили объявление - <uuid>"`
  **Пример реального ответа:**

```plaintext
{
    "status": "Сохранили объявление - e0d5f35a-ae72-4411-bfe0-5eae65ce35a2"
}
```

## Тест-кейс 2: Успешное получение объявления по его ID

**Метод:** GET /api/1/item/{id}

**Предусловие:**  
Создано объявление, например:
`1ecdb317-884a-4683-90f3-f7b0d039df25`

**Ожидаемый результат:**  
HTTP 200  
Ответ содержит поля:

- id
- createdAt
- name
- price
- sellerId
- statistics (likes, viewCount, contacts)

**Пример реального ответа:**

```plaintext
[
    {
    "createdAt": "...",
    "id": "1ecdb317-884a-4683-90f3-f7b0d039df25",
    "name": "Burger",
    "price": 300,
    "sellerId": 192348,
    "statistics": {
    "contacts": 2,
    "likes": 4,
    "viewCount": 200
    }
    }
]
```

## Тест-кейс 3: Получить все объявления по sellerID

**Метод:** GET /api/1/{sellerID}/item

**Пример sellerID:** 192348

**Ожидаемый результат:**  
HTTP 200  
Ответ — список объявлений пользователя.

**Пример реального ответа:**

```plaintext
[
    {
        "createdAt": "2025-11-25 11:13:45.403792 +0300 +0300",
        "id": "1ecdb317-884a-4683-90f3-f7b0d039df25",
        "name": "Burger",
        "price": 300,
        "sellerId": 192348,
        "statistics": {
            "contacts": 2,
            "likes": 4,
            "viewCount": 200
        }
    }
]
```

## Тест-кейс 4: Получить статистику объявления по itemID

**Метод:** GET /api/1/statistic/{itemID}

**Пример itemID:**  
`1ecdb317-884a-4683-90f3-f7b0d039df25`

**Ожидаемый результат:**  
HTTP 200  
Ответ содержит:

- contacts
- likes
- viewCount

**Пример реального ответа:**

```plaintext
[
    {
    "contacts": 2,
    "likes": 4,
    "viewCount": 200
    }
]
```
